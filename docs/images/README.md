# Documentation Images

This folder contains placeholder images for the Mission Control documentation.

## Placeholder Files

| File | Purpose | Replace With |
|------|---------|--------------|
| `dashboard-placeholder.svg` | Main dashboard screenshot | Screenshot of running Mission Control dashboard showing agent cards, task queue, and system status |
| `architecture-placeholder.svg` | System architecture diagram | Actual architecture diagram showing frontend → backend → database flow |
| `components-placeholder.svg` | Component hierarchy diagram | Diagram showing React component structure, state management, and API layer |
| `widget-examples-placeholder.svg` | Widget examples showcase | Screenshots of custom widgets you've built |

## How to Replace Placeholders

1. **Take screenshots** of your Mission Control instance
2. **Create diagrams** using tools like:
   - [Draw.io](https://draw.io)
   - [Excalidraw](https://excalidraw.com)
   - [Mermaid](https://mermaid-js.github.io/mermaid-live-editor)
   - [Figma](https://figma.com)
3. **Save as PNG or SVG** in this folder
4. **Update markdown files** to reference new images:
   ```markdown
   ![Dashboard](./images/dashboard.png)
   ```

## Recommended Image Specs

- **Screenshots**: PNG, 1280x720 or higher
- **Diagrams**: SVG preferred, or PNG with transparent background
- **File size**: Keep under 500KB for fast loading

## Current Placeholders

The current SVG placeholders are designed to:
- Show the expected content structure
- Demonstrate dark mode styling
- Provide clear guidance on what to capture

## Contributing

When contributing screenshots/diagrams:
1. Use consistent dark mode theme (#1a1a2e background)
2. Include descriptive file names
3. Update this README with new files
4. Ensure no sensitive data is visible
